# Module 03 — Dashboard (Home Screen)

## Overview
The Dashboard is Page 1 of the bottom tab navigation. It is a vertically scrolling screen with cards stacked from top to bottom. Everything updates when new data is saved — no real-time polling.

---

## Layout (top to bottom)

### 1. Top Navigation Bar (sticky)
- **Left**: Child avatar switcher (see 02-profile.md for Child Switcher component)
- **Center**: Child nickname + age string
  - EN: "Mia • 3 months 12 days"
  - FIL: "Mia • 3 buwan 12 araw"
  - ZH: "Mia • 3个月12天"
- **Right**: Two icon buttons
  - 🤖 Ate AI button → opens AI Assistant bottom sheet
  - 🔔 Notification bell → opens notifications list

### 2. Quick Stats Strip (horizontal scroll)
Four mini stat chips in a horizontal scrollable row:
- **Last Fed**: "2h 30m ago" / "2h 30m na ang nakakaraan" / "2小时30分钟前"
- **Sleep Today**: "5h 40m" / "5h 40m ngayon" / "今日睡眠5小时40分"
- **Next Vaccine**: "BCG • 12 days" / "BCG • 12 araw" / "BCG • 还有12天"
- **Weight**: latest recorded weight in kg

Chip style: white pill with colored left border, small icon, two-line text (label + value).

### 3. Growth Snapshot Card
White card, rounded corners (borderRadius 20), subtle shadow.

Contents:
- Card title: "Growth Snapshot 📈" / "Pagsusuri sa Paglaki 📈" / "成长概览 📈"
- Row of 3 stat boxes: Weight (kg) | Height (cm) | Head Circ. (cm)
- Each box: large number + unit, small "as of [date]" below
- WHO Percentile badges (color-coded pills):
  - 🟢 Green = 15th–85th (Normal)
  - 🟡 Yellow = 5th–15th or 85th–97th (Watch)
  - 🔴 Red = below 5th or above 97th (Consult Pedia)
- Mini spark-line chart: last 5 weight data points (Victory Native line chart, pink line)
- AI Summary text (1–2 sentences, generated by Claude API, shown in gray italic)
  - Example: "Si Mia ay lumalaki nang maayos! Ang timbang niya ay nasa 65th percentile ayon sa WHO."
- "View Full Analysis →" text button at bottom right

**Update trigger**: This card only refreshes when parent saves a new growth measurement. Not real-time.

### 4. Feature Icon Grid
2×3 grid of square cards (or 3×2 on wider screens).

Each card:
- Large centered emoji icon
- Label below (in chosen language)
- Subtle colored background matching category

| Icon | EN | FIL | ZH | Bg Color |
|---|---|---|---|---|
| 🍼 | Feeding Log | Tala ng Pagpapakain | 喂食记录 | #FFF3E0 |
| 😴 | Sleep Tracker | Subaybay ng Tulog | 睡眠追踪 | #F0ECFF |
| 💉 | Vaccination | Tala ng Bakuna | 疫苗记录 | #FFE4EE |
| 💊 | Vitamins | Bitamina at Gamot | 维生素与药物 | #E0F7EF |
| 🥗 | Feeding Guide | Gabay sa Pagkain | 饮食指南 | #FFF8E8 |
| 📊 | Reports | Ulat at Pagsusuri | 数据报告 | #E8F2FF |

Each card taps to open a modal/bottom sheet or push screen for that feature.

### 5. Insights Card
White card at the bottom of the scroll.

Contents:
- Title: "This Week's Summary" / "Buod ng Linggong Ito" / "本周摘要"
- Row 1: Total feeds this week + total volume (ml)
- Row 2: Total sleep hours this week + avg per day
- Row 3: Upcoming event (next vaccine or checkup)
- "View Full Reports →" text button

---

## Empty States
- If no child profile yet: show illustration + "Add your baby to get started!" prompt
- If no data logged yet: show illustration + encouraging message + "Start Tracking" button
- Each card individually handles its own empty state

---

## Pull to Refresh
- Pull down on scroll view triggers data refetch from Supabase
- Refreshes: quick stats, growth snapshot, insights card
