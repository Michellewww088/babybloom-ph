# Module 05 — Sleep Tracker (Subaybay ng Tulog / 睡眠追踪)

## Screens / Components
1. Sleep Tracker Main Screen
2. Add / Edit Sleep Entry Modal
3. Sleep Analytics View

---

## Screen 1: Sleep Tracker Main Screen

### Layout
- Header: "Sleep Tracker 😴" + "Add Entry" button
- Big center card: Active Sleep Timer (if sleep is currently in progress)
- Date filter: Today / This Week / This Month
- Summary bar: Total sleep today | Number of naps | Longest stretch
- Sleep entry list (grouped by date, newest first)

### Active Sleep Timer Card
Shows only when a sleep session is currently running:
- Large moon 🌙 animation (gentle pulse)
- "Currently sleeping..." / "Natutulog ngayon..." / "睡眠中..."
- Timer showing elapsed time: HH:MM:SS
- Big "Wake Up" button → taps to end sleep session (sun rise 🌅 animation)

---

## Screen 2: Add / Edit Sleep Entry Modal

### Mode A: Timer Mode (recommended)
- "Sleep Started" button → records start time as NOW, starts timer
- Timer runs in background (even if app is closed — use background task)
- "Wake Up" button → records end time, saves entry

### Mode B: Manual Entry
| Field | Type | Notes |
|---|---|---|
| Sleep Start | Date + Time picker | |
| Sleep End | Date + Time picker | Must be after start time |
| Duration | Auto-calculated | Shown as "Xh Ym" |

### Both Modes: Additional Fields
| Field | Type | Notes |
|---|---|---|
| Sleep Type | Radio | 🌙 Night Sleep (Gabi) / 💤 Nap (Idlip) |
| Quality (optional) | Radio | 😴 Restful (Mahimbing) / 😟 Restless (Magulo) / 😭 Frequent waking (Madalas gumising) |
| Notes | Text input | Optional, e.g. "slept in stroller" / "natulog sa bisita" |

---

## Screen 3: Sleep Analytics View

### Summary Stats Row
- Total sleep this week (hours + minutes)
- Average per day
- Longest single sleep stretch this week
- Number of night sleeps vs. naps

### Sleep Timeline Chart (daily view)
- 24-hour horizontal bar chart (like a Gantt chart)
- Each sleep block shown as a colored bar:
  - 🌙 Night sleep = dark blue/purple bar
  - 💤 Nap = light lavender bar
- X-axis: 00:00 to 23:59
- Y-axis: last 7 days
- Tap a bar to see: start time, end time, duration, quality

### Weekly Bar Chart
- Bar per day showing total sleep hours
- Horizontal reference line: WHO recommended hours for baby's age
  - 0–3 months: 14–17 hours
  - 4–11 months: 12–16 hours
  - 1–2 years: 11–14 hours

### AI Sleep Insight
One paragraph generated by Claude API:
- Example (FIL): "Si [nickname] ay natulog ng average na 14.5 oras ngayon. Ayon sa WHO, normal para sa isang 3-buwang sanggol ang 14–17 oras ng tulog bawat araw. Magaling! 🌙"

---

## Data Stored (sleep_logs table)
- started_at: timestamp (required)
- ended_at: timestamp (nullable — null means currently sleeping)
- sleep_type: 'night' | 'nap'
- quality: 'restful' | 'restless' | 'frequent_waking' | null
- notes: string | null

## Edge Cases
- If ended_at is null and it's been > 24 hours, show a prompt: "Is [nickname] still sleeping? Please update the sleep entry."
- Overlapping sleep entries: show validation error, do not allow
- Manual entry where end < start: show validation error
